# Deep Work Tracker

Простой сервис для отслеживания сессий глубокой работы с визуализацией как на GitHub.

## Запуск локально

1. Убедитесь, что установлен Node.js.
2. Откройте терминал в этой папке.
3. Установите зависимости (если еще нет):
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   npm start
   ```
5. Откройте в браузере: [http://localhost:3000](http://localhost:3000)

## Развертывание на сервере

Этот проект готов к развертыванию на любой платформе, поддерживающей Node.js (например, Railway, Render, DigitalOcean, Heroku).

Для постоянного хранения данных используется SQLite база данных (`deepwork.db`). При развертывании убедитесь, что ваш хостинг поддерживает персистентные диски (Persistent Volume), иначе база данных будет сбрасываться при перезагрузке сервера.

Если вам нужен простой вариант "в облаке" без настройки сервера, рекомендую использовать сервисы вроде **Railway** или **Render**, подключив этот репозиторий.
